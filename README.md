# pro-worldcup26-notes

本仓库用于归档与发布多个独立的 HTML 页面。页面内容涵盖赛事记录、数据整理、视觉呈现等方向，不针对特定域名或网站。仓库维护者定期更新，供参考与学习使用。

## 目录说明

```
├── pages/            # 存放所有 HTML 页面文件
├── assets/           # 页面引用的图片、样式、脚本等资源
├── index.html        # 仓库首页（可选）
└── README.md         # 本文件
```

- **pages/**：每个 `.html` 文件为一个独立页面，命名按内容或日期区分。
- **assets/**：资源文件按类型（如 `img/`、`css/`、`js/`）存放，便于页面引用。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 页面不依赖外部服务或特定域名，内容以静态形式呈现。
- 归档内容可能包含实验性设计、数据可视化、信息整理等类型。
- 部分页面可能未做完整适配，建议在桌面端查看。

## 维护说明

- 仓库不定期更新，新增页面或修正内容。
- 不保证所有页面长期兼容所有浏览器。
- 如发现页面显示异常或内容错误，欢迎提交 Issue 反馈。
- 本仓库仅供个人记录与存档，不做商业用途。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 在浏览器中直接打开 `pages/` 目录下的 `.html` 文件。
3. 也可通过 GitHub Pages 等服务部署后访问（需自行配置）。
